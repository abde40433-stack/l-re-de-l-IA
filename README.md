# l-re-de-l-IA
L'Intelligence Artificielle n'est plus de la science-fiction. Elle est présente dans nos téléphones, nos voitures et nos hôpitaux. Cette application interactive transforme le rapport source en une expérience exploratoire pour comprendre ce qu'est l'IA, comment elle a évolué (ses "hivers" et son âge d'or) et comment elle redéfinit notre réalité.
